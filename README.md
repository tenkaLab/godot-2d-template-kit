# godot-2d-template-kit

Godot 4 / 2D / GDScript 向けのゲーム制作テンプレート集です。

## 目的

ゲームエンジンそのものを作るのではなく、Godotで2Dゲームを作り始めるための基礎構造を整えることを目的としています。

## 初期テンプレート候補

- minimal-game-flow
  - Title → Main → Result → Title の最小シーン遷移テンプレート

- flag-manager-basic
  - ストーリーフラグ、イベントフラグ管理

- save-load-basic
  - JSONベースのセーブ/ロード

- dialogue-basic
  - JSON会話データ読み込み

- item-registry-basic
  - アイテム定義、所持状態管理

## 今後の予定

最初はテンプレート集として小さく始めます。
将来的には、テンプレ調整GUI、import/export、プロジェクト診断ツールなどへの発展も視野に入れています。
